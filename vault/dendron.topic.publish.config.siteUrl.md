- type: `string`
- default: N/A
- required: `false`

## Description
{{fm.desc}}
All links will be prefixed with this URL.


## Example
If you are publishing with GitHub Pages, the format is `https://{username}.github.io/{repoName}`

```yml
  publishing:
    siteUrl: https://johndoe.github.io/pkm
```
