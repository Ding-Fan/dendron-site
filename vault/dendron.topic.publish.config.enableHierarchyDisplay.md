- type: `boolean`
- default: `true` 
- required: `false`

## Description
{{fm.desc}}

The text used for the title of the block can be set by [[hierarchyDisplayTitle|dendron.topic.publish.config.hierarchyDisplayTitle]]

## Example

```yml
  publishing:
    enableHierarchyDisplay: true
    hierarchyDisplayTitle: Children
```
